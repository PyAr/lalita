Lalita

¡Welcome to Lalita! - ¡Bienvenidos a Lalita!

Who is Lalita?

Lalita is an IRC bot where new functionality is simple to create by just adding easy-to-write plugins. The goal is to make it very easy to implement what we need in a bot.

How do I install it?

Two ways, mainly:

  • .tar.gz: Just source code, to install it with setup.py install, see the README for more details.
  • In Debian or Ubuntu you can subscribe to the PPA and automatically receive all the updates: sudo apt-add-repository ppa:laliputienses/lalita-ppa (more info here)
