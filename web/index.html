Lalita

¡Bienvenidos a Lalita! - Welcome to Lalita!

¿Quién es Lalita?

Lalita es un bot de IRC al que de forma sencilla se le pueden agregar plugins con nuevas funcionalidades. El objetivo es que sea muy fácil implementar lo que uno necesite en un bot.

¿Cómo lo instalo?

Hay dos maneras principalmente:

  • .tar.gz: Código fuente hecho y derecho, para instalar con setup.py install, ver el README que trae para más detalles.
  • En Debian o Ubuntu pueden suscribirse al PPA y recibir las actualizaciones automáticamente: sudo apt-add-repository ppa:laliputienses/lalita-ppa (más info acá)

¿Cómo lo aprendo?

Acá está el tutorial, un documento que explica cómo usar Lalita, como instalarlo, probarlo, configurarlo, etc.
